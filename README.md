# **Learning of Node Streams using version 17 of the nodejs**

The intention of this project is to study the new features of nodejs in version 17, such as node-stream.


### REQUIRED
  - Node version 17

### RUN
`git clone [this repository]`

```
    cd node-stream 
```


```
    npm install 
```


```
    npm run server
```


```
    npm run client
```
